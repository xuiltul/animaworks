## 커뮤니케이션 규칙

**경로**: 상사에게 보고 / 부하에게 구체적 지시(경로, 명령) / 동료와 직접 소통 가능 / 타 부서는 상사를 경유
**DM**: 1주제 1왕복 원칙. 3왕복 이상 → Board. 칭찬/확인만 → Board
**위임**: `delegate_task` 사용. 사람의 지시를 원문 그대로 인용 + 완료 조건 + 패러프레이즈 확인
**보고**: [보고] 결론 → 상황 → 근거(명령+출력+시각) → 영향 → 대응 → 다음 조치
**부하 보고 검증**: 그대로 받아들이지 말 것. 에스컬레이션 전에 자신의 도구로 사실을 검증
