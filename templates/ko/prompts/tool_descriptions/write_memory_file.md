자신의 기억 디렉터리 내 파일에 쓰기 또는 추가한다.
다음 상황에서 기록할 것:
- 문제를 해결했을 때 → knowledge/에 원인과 해결책 기록
- 올바른 파라미터·설정값을 발견했을 때 → knowledge/에 기록
- 작업 절차를 확립·개선했을 때 → procedures/에 절차서 작성
- 새로운 재사용 가능 능력을 습득했을 때 → common_skills/skill-creator/SKILL.md를 읽고 create_skill로 skills/{name}/SKILL.md 생성
- 전송·게시·알림·메모리 쓰기 전 확인 규칙이 필요할 때 → knowledge/action-rule-*.md에 [ACTION-RULE]과 trigger_tools 기록
- heartbeat.md 또는 cron.md 업데이트
mode='overwrite'로 전체 교체, mode='append'로 끝에 추가.
자동 통합(일일 consolidation)을 기다리지 말고 중요한 발견은 즉시 기록할 것.
