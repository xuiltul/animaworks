# Market Data Collector — 품질 체크리스트

Market Data Collector가 데이터 수집 품질을 보장하기 위한 체크리스트.

---

## A. 데이터 품질(Data Quality)

- [ ] 출처 URL·API 이름·데이터 제공처가 기록되어 있는가
- [ ] 취득 일시(타임스탬프)가 정확한가
- [ ] 데이터 신선도가 분석 목적에 충분한가(오래된 데이터가 아닌가)
- [ ] 단위·통화가 명시되어 있는가
- [ ] 추측이나 추정으로 보충한 값이 없는가

---

## B. 완전성(Completeness)

- [ ] Director가 지정한 전 데이터 항목을 수집했는가
- [ ] 시계열 데이터에 결손 기간이 없는가
- [ ] 취득 불가 항목을 명시했는가(사유 포함)
- [ ] 복수 소스 교차 검증을 실시했는가(가능한 경우)

---

## C. 정확성(Accuracy)

- [ ] 복수 소스 간 모순이 있을 때 전 소스 값을 병기했는가
- [ ] 통화 환산이 필요한 경우 환산 레이트와 기준일이 기록되어 있는가
- [ ] 과거 수집 결과와의 일관성을 확인했는가(갑작스러운 대폭 변동이 없는가)
- [ ] 이상값이 있을 때 소스 측 문제인지 데이터 자체 변동인지 구별했는가

---

## D. 보고 품질

- [ ] 데이터가 Director 분석에 쓰기 쉬운 형식으로 구조화되어 있는가
- [ ] 수집 건수·기간·소스가 명시되어 있는가
- [ ] 복수 소스 간 모순을 보고했는가
- [ ] 데이터의 제약(타임 랙·커버리지·정확도 한계)이 기재되어 있는가
