# Data Analyst — 품질 체크리스트

다단계 검증 프로세스에 따라 Data Analyst가 데이터 추출 품질을 보장하기 위한 체크리스트.

---

## A. 추출 정확성(Extraction Accuracy)

- [ ] 모든 수치를 원전과 대조했는가
- [ ] 제1단계(기계적 추출)와 제2단계(구조 확인)를 모두 실시했는가
- [ ] 합계값·소계가 개별 항목 합산과 일치하는가
- [ ] 자릿수·단위가 정확한가(천엔/백만엔/엔 혼재 없음)
- [ ] 추정값이나 대략값을 사용하지 않았는가
- [ ] 불명 항목을 추측으로 채우지 않았는가(빈칸으로 두었는가)

---

## B. 구조 일관성(Structural Integrity)

- [ ] 계정과목 체계와 일치하는가
- [ ] 필수 항목(자산·부채·자본·수익·비용의 주요 과목)이 결손되지 않았는가
- [ ] 기간(연월일) 지정이 정확한가
- [ ] 전기 데이터와 구조적으로 일관되는가(과목 추가·삭제·명칭 변경을 파악했는가)
- [ ] 복수 소스 간 데이터에 모순이 없는가

---

## C. 출처 기록(Source Documentation)

- [ ] 모든 수치에 출처(원전 파일명·페이지 번호 등)가 기록되어 있는가
- [ ] 원전 데이터의 취득일·버전이 기록되어 있는가
- [ ] 원전으로부터의 전기 방법(직접 입력 / 도구 추출)이 기록되어 있는가

---

## D. 보고 품질

- [ ] 추출 건수가 기재되어 있는가
- [ ] 검증 결과(합격 / 차이 있음)가 명시되어 있는가
- [ ] 차이나 오류가 발견된 경우 상세가 기재되어 있는가
- [ ] 불명 항목 목록과 이유가 기재되어 있는가
- [ ] Director가 분석에 사용할 수 있는 형식으로 구조화되어 있는가
