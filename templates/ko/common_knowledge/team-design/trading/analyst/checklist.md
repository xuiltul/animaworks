# Market Analyst — 품질 체크리스트

machine 사용 여부와 관계없이 Market Analyst가 시그널과 데이터 품질을 보장하기 위한 체크리스트.

---

## A. 시그널 품질

- [ ] 시그널에 통계적 근거(p값, 신뢰구간, 표본 크기 등)가 붙어 있는가
- [ ] 확도가 3단계(고확도/중확도/참고 수준)로 명시되어 있는가
- [ ] 위양성률(False Positive Rate)을 평가했는가
- [ ] 과거 유사 시그널의 적중률을 확인했는가
- [ ] 복수의 독립 데이터 소스로 뒷받침이 되는가
- [ ] 장세 레짐(추세/횡보/변동성)을 고려한 분석인가

---

## B. 데이터 품질

- [ ] 사용한 데이터 소스의 신뢰성을 확인했는가
- [ ] 데이터의 신선도(최종 갱신 시각)를 확인했는가
- [ ] 결손값·이상치 처리 방법을 명시했는가
- [ ] 데이터 취득 기간이 시그널 시간축에 대해 충분한가
- [ ] 복수 데이터 소스 간에 모순이 없는지 확인했는가

---

## C. 모델 검증

- [ ] 백테스트 vs 라이브 성과 괴리를 확인했는가
- [ ] 워크포워드 분석으로 과적합 징후가 없는지 검증했는가
- [ ] 레짐 변화(시장 환경의 구조적 변화)에 대한 모델 견고성을 평가했는가
- [ ] 파라미터 민감도 분석을 수행했는가(미세 변동으로 결과가 크게 바뀌지 않는지)
- [ ] 서바이버십 바이어스가 제거되어 있는가
- [ ] 룩어헤드 바이어스가 제거되어 있는가

---

## D. 리포트 품질

- [ ] market-analysis.md 형식을 따르는가
- [ ] 실행 요약이 결론을 간결히 전달하는가
- [ ] 전 시그널에 확도와 통계적 근거가 붙어 있는가
- [ ] 전제 조건과 한계가 명시되어 있는가
- [ ] 리스크 요인이 열거되어 있는가
- [ ] 「고확도」 시그널에 충분한 뒷받침이 있는가(뒷받침이 약하면 확도를 낮춘다)

---

## E. machine 출력 검증

### machine에 데이터 수집·계산을 위임한 경우

- [ ] 계산 결과를 독립된 방법으로 검산했는가(샘플링 체크 가능)
- [ ] 이상치·외삽값이 올바르게 처리되었는가
- [ ] 데이터 취득 범위·기간이 지시대로인가
- [ ] machine이 생성한 통계값의 단위·스케일이 맞는가
- [ ] 출력에 Analyst 본인의 판단을 더한 뒤 market-analysis.md를 완성했는가
