# Strategy Director — 품질 체크리스트

machine 사용 여부와 관계없이 Strategy Director가 전략 품질과 PDCA 회전을 보장하기 위한 체크리스트.

---

## A. P&L 모니터링 Gate

- [ ] 현재 누적 P&L을 확인했는가
- [ ] 드로다운이 임계 `{max_drawdown_pct}` 이내임을 확인했는가
- [ ] 드로다운이 임계에 근접(80% 이상)한 경우 대응책을 검토했는가
- [ ] 손익 트렌드(개선/횡보/악화)를 평가했는가
- [ ] 전 전략 합산 포트폴리오 리스크를 확인했는가

---

## B. 전략 품질 Gate

- [ ] 백테스트 in-sample과 out-of-sample에서 성과에 큰 괴리가 없는가
- [ ] 파라미터 민감도 분석을 수행하고, 파라미터 미세 변동으로 결과가 크게 바뀌지 않는지 확인했는가
- [ ] 엣지 지속성을 평가했는가(최근 승률 vs 전체 기간 승률)
- [ ] 슬리피지·수수료가 현실적인 값으로 반영되어 있는가
- [ ] 서바이버십 바이어스가 제거되어 있는가
- [ ] 룩어헤드 바이어스가 제거되어 있는가

---

## C. 리스크 관리 Gate

- [ ] 전 포지션 크기가 `{position_limit}` 이내인가
- [ ] 레버리지가 `{leverage_limit}` 이내인가
- [ ] 상관 집중 리스크를 확인했는가(동일 방향 복수 포지션)
- [ ] 유동성 리스크를 확인했는가(호가 얇은 종목 포지션)
- [ ] 손절 규칙이 전 전략에 설정되어 있는가
- [ ] 최대 동시 보유 포지션 수를 확인했는가

---

## D. PDCA 진행 Gate

- [ ] 전회 Check 지적 사항이 전건 Act에 반영되었는가
- [ ] performance-tracker가 최신 상태로 갱신되어 있는가(silent drop 0)
- [ ] ops-issue-tracker의 미해소 항목을 확인했는가
- [ ] PDCA Act에서 내린 판단(계속/조정/정지)에 근거가 명시되어 있는가
- [ ] 다음 Check 예정일이 설정되어 있는가

---

## E. 위임 시 품질

### Engineer / Analyst에 의뢰하기 전에

- [ ] strategy-plan.md의 `status: approved`를 확인했는가
- [ ] 리스크 파라미터가 명시되어 있는가
- [ ] 완료 조건이 구체적이고 검증 가능한가
- [ ] 기한(deadline)을 설정했는가
- [ ] Performance Tracker에서의 인계 사항이 strategy-plan.md에 포함되어 있는가

### Auditor 피드백 대응

- [ ] Auditor 지적 사항을 전건 확인했는가
- [ ] P&L 괴리 지적에 대해 원인 분석을 수행했는가
- [ ] 운용 건전성 문제에 대해 대응책을 결정했는가
- [ ] performance-tracker / ops-issue-tracker를 최종 갱신했는가
