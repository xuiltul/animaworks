# Legal Director — 品質チェックリスト

machine 使用有無に関わらず、Legal Director が成果物の品質を担保するためのチェックリスト。

---

## A. 前回監査との整合性（Carry-forward Gate）

- [ ] 前回監査の Critical 指摘が全件リストアップされているか
- [ ] 各 Critical 指摘に対して「解消 / 部分解消 / 未修正 / 悪化」のステータスが付いているか
- [ ] 「部分解消」の項目に残存リスクの評価があるか
- [ ] 前回の High 指摘のうち、今回の修正版で変更がなかった項目が列挙されているか
- [ ] 前回の Medium 指摘で、修正版で悪化した項目がないか確認したか
- [ ] carry-forward tracker のステータスが全件更新されているか（silent drop ゼロ）

---

## B. 条項網羅性（Clause Coverage）

- [ ] 契約書の全条項を通読したか（修正箇所だけでなく全体）
- [ ] 修正された条項以外に、前回版から変更された箇所がないか確認したか
- [ ] 空欄項目（●●）の残存を確認したか
- [ ] 定義条項（第1条相当）の変更有無を確認したか
- [ ] 付属書・別紙の有無と内容を確認したか

---

## C. リスク評価の根拠（Risk Calibration）

各リスク項目について:

- [ ] リスク評価（Critical/High/Medium/Low）に法的根拠（条文・判例・ガイドライン）があるか
- [ ] 「受容可」「追加交渉不要」と判定した項目に、判定理由が明記されているか
- [ ] 「業界標準」と記載した箇所に具体的な根拠（判例・実務書・統計）があるか
- [ ] 前回より低いリスク評価に変更した項目に、変更理由が明記されているか
- [ ] 「相手方が拒否する理由がない」等の推測的表現で根拠を省略していないか

---

## D. 契約固有チェック

対象契約の種類に応じて、以下から該当する項目を確認する:

### D-1. 株式譲渡契約（SPA）/ M&A

- [ ] 知的財産権の帰属（Background IP / Foreground IP）
- [ ] 補償条項（上限額・免責額・請求期限・計算方法・文言リスク）
- [ ] 表明保証の範囲と存続期間
- [ ] 競業避止・勧誘禁止の範囲と期間
- [ ] クロージング条件（支払方法・時期・確認手段）
- [ ] 解除条件と解除後の権利義務存続
- [ ] 対価の算定方法と調整条項

### D-2. 秘密保持契約（NDA）/ 一般契約共通

- [ ] 秘密保持義務の範囲・存続・除外規定
- [ ] 完全合意条項と既存契約の関係
- [ ] 努力義務の水準（best efforts / reasonable efforts / commercially reasonable）
- [ ] 当事者の表記の正確性（法人形態・住所・代表者名）
- [ ] 管轄裁判所・準拠法
- [ ] 損害賠償の範囲（直接損害・間接損害・逸失利益）
- [ ] 反社会的勢力排除条項

---

## E. 出力品質

- [ ] 事実の誤記がないか（人名・会社名・法人形態・日付・金額）
- [ ] 推奨アクションが全リスク項目に付いているか
- [ ] 交渉優先度が明記されているか
- [ ] 署名前に解決すべき事項と署名後でよい事項が区分されているか
- [ ] 非法務メンバーにも理解できる平易な説明になっているか

---

## F. 委譲時の品質

### Verifier / Researcher への依頼前に

- [ ] audit-report.md の `status: reviewed` を確認したか
- [ ] analysis-plan.md（分析計画）が添付されているか
- [ ] carry-forward tracker が最新ステータスに更新されているか
- [ ] Verifier / Researcher が検証に必要な情報（契約書本文・前回レポート）がアクセス可能か
- [ ] 期限（deadline）を設定したか

### フィードバック対応

- [ ] Verifier / Researcher からのフィードバックを全件確認したか
- [ ] 楽観バイアスの指摘に対して再評価を実施したか
- [ ] Researcher の裏付け結果を audit-report.md に反映したか
- [ ] carry-forward tracker の最終ステータスを更新したか
