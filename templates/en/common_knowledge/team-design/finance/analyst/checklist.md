# Data Analyst — quality checklist

Checklist for Data Analyst to ensure extraction quality following the multi-stage verification process.

---

## A. Extraction accuracy

- [ ] Every figure reconciled to authoritative source
- [ ] Both stage 1 (mechanical) and stage 2 (structure) completed
- [ ] Subtotals and totals match sum of lines
- [ ] Digits and units correct (no mix of thousands / millions / currency units)
- [ ] No estimates or rough figures used
- [ ] Unknown fields left blank, not guessed

---

## B. Structural integrity

- [ ] Aligned with chart of accounts structure
- [ ] No missing required lines (major asset, liability, equity, revenue, expense accounts)
- [ ] Periods (dates) accurate
- [ ] Structural consistency with prior period (adds, removes, renames understood)
- [ ] No contradictions across sources

---

## C. Source documentation

- [ ] Every figure has provenance (source file name, page, etc.)
- [ ] Source retrieval date / version recorded
- [ ] Method of capture recorded (manual entry vs. tool extraction)

---

## D. Reporting quality

- [ ] Extraction counts stated
- [ ] Verification outcome clear (pass / variances)
- [ ] If variances or errors, details documented
- [ ] List of unknown fields and reasons
- [ ] Structured in a form Director can use for analysis
