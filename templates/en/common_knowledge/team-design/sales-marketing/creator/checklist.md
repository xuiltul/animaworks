# Marketing Creator — quality checklist

Quality checklist for Marketing Creator regardless of machine usage.

---

## A. content-plan compliance

- [ ] Purpose from content-plan.md reflected
- [ ] Content appropriate for specified target (persona / industry / role)
- [ ] Key messages present in the body
- [ ] Structure instructions (outline / word count) followed
- [ ] Format matches specified content type (blog / email / whitepaper, etc.)

---

## B. Brand Voice compliance

- [ ] Tone appropriate for content type (professional / casual / formal, etc.)
- [ ] No prohibited expressions (exaggeration, competitor attacks, etc.)
- [ ] Terminology unified (no inconsistencies)
- [ ] Persona-specific style guide followed (if configured)

---

## C. Funnel alignment

- [ ] Content depth appropriate for funnel stage (TOFU/MOFU/BOFU)
- [ ] CTA appropriate for stage (TOFU: awareness / MOFU: consideration / BOFU: decision)
- [ ] Path to next funnel stage included

---

## D. Compliance first filter

- [ ] No obvious exaggerated claims
- [ ] No unsupported "No. 1" or "industry-leading" claims
- [ ] No inappropriate health/medical efficacy claims (if applicable)
- [ ] Compliance notes from content-plan.md followed
- [ ] Concerns flagged in draft-content.md if any

---

## E. Output quality

- [ ] No typos or grammatical errors
- [ ] URLs correct (if included)
- [ ] Image/diagram instructions clear (if needed)
- [ ] Heading structure logical
- [ ] Appropriate length (within ±20% of specified word count)
